# Rack Designer

**Rack Designer** is a powerful desktop application for designing AV racks. Easily drag-and-drop gear, track Rack Units (RU), and monitor total weight for your projects.

    - *Note: You may need to right-click and select "Open" the first time due to macOS security settings.*

- **Drag-and-Drop Interface**: Easily place equipment in your rack.

- **Dynamic Rack Selection**: Manage multiple racks in a single project with horizontal scrolling selection.
